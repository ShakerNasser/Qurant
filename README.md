## Qurant- Work sample API

This repository contains a sample implementation of a RESTful API using Laravel for managing absence notifications to a payroll system.

**Prerequisites**
- PHP
- Composer
- Laravel CLI
- MySQL or SQLite database
- (Docker- devcontainer)

### Source
